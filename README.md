# دليل السفر إلى السعودية | Travel Guide to Saudi Arabia

A fully responsive bilingual (Arabic/English) landing page for Saudi Arabia travel guide.

## Project Structure

```
.
├── index.html          # Main HTML file
├── style.css           # Stylesheet with RTL/LTR support
├── script.js           # JavaScript for interactivity
├── assets/             # Assets folder
│   └── logo-placeholder.svg
└── README.md           # This file
```

## Features

- ✅ Fully responsive design
- ✅ Bilingual support (Arabic/English)
- ✅ RTL/LTR text direction support
- ✅ Language switcher with localStorage persistence
- ✅ Smooth scrolling navigation
- ✅ Accordion-style FAQ section
- ✅ Scroll animations
- ✅ Modern design with VisitSaudi color scheme
- ✅ Mobile-friendly navigation menu

## Sections

1. **Header** - Navigation with language switcher
2. **Hero Section** - Main banner with CTA button
3. **About Saudi Arabia** - Key highlights about the Kingdom
4. **Travel Guide** - Planning tips and best times to visit
5. **Transportation** - Getting around Saudi Arabia
6. **Visa & FAQs** - Visa information and frequently asked questions
7. **Registration Form** - Placeholder for Zoho form embed
8. **Footer** - Navigation links and social media placeholders

## How to Use

1. Open `index.html` in a web browser
2. Use the language switcher in the header to toggle between Arabic and English
3. The language preference is saved in localStorage
4. Replace the Zoho form placeholder with actual embed code when provided

## Adding Zoho Form

To add the Zoho form embed code, replace the content in the `zoho-form-placeholder` div or use the `addZohoForm()` function in the browser console:

```javascript
addZohoForm('your-zoho-embed-code-here');
```
